# ts-chat
A Chat web-app running directly from TypeScript
with Socket.io, TS-Node, Browserify and ExpressJS

```
npm install
```
Run (It's restarted when source code changes)
```
npm run start
```
Clone it to build awesome stuff with it ❤️
